# Baseline Revenue Tracker

A React-based visualization tool for tracking and analyzing traffic distribution across different channels.

## Getting Started

1. Clone the repository
```bash
git clone https://github.com/your-username/baseline-revenue-tracker.git
```

2. Install dependencies
```bash
cd baseline-revenue-tracker
npm install
```

3. Start the development server
```bash
npm start
```

4. Build for production
```bash
npm run build
```

## Features

- Interactive stacked area chart showing traffic distribution
- Detailed channel breakdown
- Quarterly progression tracking
- Responsive design

## Technology Stack

- React
- Recharts
- Tailwind CSS
- React Testing Library

## Configuration

Edit the data in `src/data/revenueData.js` to update:
- Traffic distribution targets
- Channel breakdown
- Session numbers
